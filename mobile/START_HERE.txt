🚀 INICIO RÁPIDO (5 PASOS)

1️⃣  npm install
    └─ Instala todas las dependencias

2️⃣  Edita: src/supabaseClient.js
    ├─ Agrega SUPABASE_URL
    └─ Agrega SUPABASE_KEY

3️⃣  Copia SQL de SQL_QUERIES.sql al Supabase SQL Editor
    ├─ Secciones 1-4 (tablas, RLS, usuario, viajes)
    └─ ⚠️ Reemplaza USER_ID_AQUI con el ID real

4️⃣  npm run dev
    └─ Escanea QR con Expo Go

5️⃣  Login: pilot1@example.com / password123
    └─ ¡Deberías ver tus viajes!
